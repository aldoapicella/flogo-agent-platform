# v0.2.0

This release adds release-managed startup updates for packaged installs.

- checks GitHub Releases on startup for released builds
- shows curated plus generated release notes before applying updates
- supports `flogo-agent update check` and `flogo-agent update apply`
- updates the managed `flogo` companion binary alongside the agent when the agent owns that install
- records updater state in user config and exposes update status in `flogo-agent doctor`
- publishes a release manifest and requires curated release summaries in the release workflow
